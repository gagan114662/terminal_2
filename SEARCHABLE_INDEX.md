### To run TermNet:
- `python -m termnet.main` or `./run.sh`

- `./verify_phase3_build.sh`

- `python gpu_monitor.py`

### To run tests:
- `pytest` (from TermNet directory)

### To build Docker:
- `docker-compose up`

### To verify claims:
- `python verify_termnet_claims.py`
